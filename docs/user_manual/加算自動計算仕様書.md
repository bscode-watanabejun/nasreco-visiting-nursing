## 概要

本システムでは、訪問記録を登録・更新する際に、自動的に加算項目を判定し、適用可能な加算を計算します。

### 加算が自動計算されるタイミング

- **訪問記録を作成したとき**
- **訪問記録を更新したとき**
- **月次レセプトを再計算したとき**

### 加算の確認方法

訪問記録の詳細画面で「適用加算」セクションに、自動計算された加算が表示されます。

---

## 加算自動計算の仕組み

### 計算の流れ

1. 訪問記録の情報（訪問日、訪問時間、患者情報など）を取得
2. 施設の設定情報（24時間対応体制など）を取得
3. 加算マスタから、訪問日に有効な加算項目を取得
4. 各加算項目の適用条件を順番にチェック
5. 条件を満たした加算を「適用加算」として記録
6. 訪問記録の点数に加算点数を加算

### 適用条件の判定に使用される情報

- **訪問記録画面**の入力項目
  - 訪問日
  - 訪問開始時刻・終了時刻
  - 退院日当日の訪問（チェックボックス）
  - 新規計画初回訪問（チェックボックス）
  - 多職種連携記録あり（チェックボックス）
  - ターミナルケア（チェックボックス）
  - 複数回訪問の理由
  - 緊急訪問の理由
  - 長時間訪問の理由
  - 専門的ケアの種類
  - 担当看護師

- **患者情報画面**の入力項目
  - 生年月日（年齢計算に使用）
  - 死亡日
  - 死亡場所
  - 建物情報
  - 特別管理項目

- **施設設定画面**の設定項目
  - 24時間対応体制加算（基本）
  - 24時間対応体制加算（看護業務負担軽減）
  - 緊急時訪問看護加算（I）
  - 緊急時訪問看護加算（II）

## 医療保険の加算項目

| 加算名 | 適用条件 | 点数 | 備考 |
|--------|---------|------|------|
| 緊急訪問看護加算 | 訪問記録画面の「緊急訪問の理由」に入力があること | 月14日目まで: 2,650円<br>月15日目以降: 2,000円 | 同じ月内の緊急訪問回数で判定（全訪問回数ではない） |
| 長時間訪問看護加算 | 訪問記録画面の「訪問開始時刻」と「訪問終了時刻」から計算した訪問時間が90分を超えること | 5,200円 | 訪問時間は開始時刻から終了時刻までの時間で計算 |
| 難病等複数回訪問加算 | 訪問記録画面の「複数回訪問の理由」に入力があり、同じ日に2回目以降の訪問であること | 1日に2回・同一建物内1-2人: 4,500円<br>1日に2回・同一建物内3人以上: 4,000円<br>1日に3回以上・同一建物内1-2人: 8,000円<br>1日に3回以上・同一建物内3人以上: 7,200円 | 同一建物内の人数は、患者情報の建物情報から自動判定 |
| 乳幼児加算（基本） | 患者情報画面の「生年月日」から計算した年齢が6歳未満であること | 1,300円 | 訪問日の時点での年齢で判定 |
| 乳幼児加算（基準告示第2の4該当） | 患者情報画面の「生年月日」から計算した年齢が6歳未満で、基準告示第2の4該当であること | 1,800円 | 訪問日の時点での年齢で判定 |
| 夜間・早朝訪問看護加算 | 訪問記録画面の「訪問開始時刻」が以下の時間帯であること<br>・夜間: 18時00分～22時00分<br>・早朝: 6時00分～8時00分 | 2,100円 | 時間帯の判定は訪問開始時刻で行う。日本時間（JST）で判定 |
| 深夜訪問看護加算 | 訪問記録画面の「訪問開始時刻」が以下の時間帯であること<br>・深夜: 22時00分～6時00分 | 4,200円 | 時間帯の判定は訪問開始時刻で行う。日本時間（JST）で判定 |

| 24時間対応体制加算（看護業務負担軽減） | 施設設定画面で「24時間対応体制加算（看護業務負担軽減）」が有効で、看護業務負担軽減の取り組みが2項目以上設定されていること | 6,800円 | 施設設定は全訪問記録に適用 |
| 訪問看護ターミナルケア療養費1 | 患者情報画面の「死亡日」が設定され、訪問記録画面の「訪問日」が死亡日と同じ日で、患者情報画面の「死亡場所」が「在宅」または「特別養護老人ホーム等」で、訪問記録画面の「ターミナルケア」にチェックが入り、死亡日前14日以内にターミナルケアの訪問が2回以上あること | 25,000円 | 死亡日の訪問にのみ算定可能。死亡日前14日以内の期間は死亡日を含めて計算 |
| 訪問看護ターミナルケア療養費2 | 患者情報画面の「死亡日」が設定され、訪問記録画面の「訪問日」が死亡日と同じ日で、患者情報画面の「死亡場所」が「特別養護老人ホーム等」で、訪問記録画面の「ターミナルケア」にチェックが入り、死亡日前14日以内にターミナルケアの訪問が2回以上あること | 10,000円 | 死亡日の訪問にのみ算定可能。死亡日前14日以内の期間は死亡日を含めて計算 |
| 専門管理加算 | 訪問記録画面の「専門的ケアの種類」が選択され、訪問記録画面の「担当看護師」が設定され、担当看護師の「専門資格」に実施した専門的ケアに対応する資格があること（緩和ケア→「緩和ケア」の資格、褥瘡ケア→「褥瘡ケア」の資格、特定行為→「特定行為研修」の資格）。同じ月内でこの加算が算定されていないこと（月1回まで） | 2,500点 | 専門資格は看護師情報で設定。専門的ケアの種類と専門資格が一致している必要がある |
| 退院時支援指導加算（基本） | 訪問記録画面の「退院日当日の訪問」にチェックが入っていること | 6,000円 | - |
| 退院時支援指導加算（長時間） | 訪問記録画面の「退院日当日の訪問」にチェックが入り、訪問記録画面の「訪問開始時刻」と「訪問終了時刻」から計算した訪問時間が90分を超えること | 8,400円 | 基本と長時間は同じ訪問記録では併算定不可 |
| 退院時共同指導加算 | 訪問記録画面の「退院日当日の訪問」にチェックが入り、訪問記録画面の「多職種連携記録あり」にチェックが入っていること | 8,000円 | - |
| 特別管理指導加算 | 訪問記録画面の「退院日当日の訪問」にチェックが入り、同じ訪問記録で「退院時共同指導加算」が算定されていること | 2,000円 | 退院時共同指導加算と併せて算定する必要がある |
| 初回加算I | 訪問記録画面の「新規計画初回訪問」にチェックが入っていること | 3,000円 | 新規計画初回訪問は、訪問看護計画書を作成した後の初回訪問を指す |
| 初回加算II | 訪問記録画面の「新規計画初回訪問」にチェックが入り、訪問記録画面の「訪問開始時刻」と「訪問終了時刻」から計算した訪問時間が90分を超えること | 4,200円 | 初回加算IとIIは同じ訪問記録では併算定不可 |

## 加算の併算定ルール

### 併算定可能な加算

基本的に、異なる種類の加算は併算定（同時に算定）できます。

例:
- 緊急訪問看護加算 + 長時間訪問看護加算
- 時間帯別加算 + 24時間対応体制加算
- 退院時支援指導加算 + 退院時共同指導加算

### 併算定できない加算

一部の加算は、併算定できない組み合わせがあります。

例:
- 退院時支援指導加算（基本）と退院時支援指導加算（長時間）は、同じ訪問記録では併算定できません
- 初回加算Iと初回加算IIは、同じ訪問記録では併算定できません

### 加算の評価順序

加算は、加算マスタの「表示順」に従って評価されます。先に評価された加算が優先されます。

## よくある質問

### Q1. 加算が適用されないのはなぜですか？

**A.** 以下の点を確認してください:

1. **訪問記録の入力項目が正しく設定されているか**
   - チェックボックスにチェックが入っているか
   - 訪問開始時刻・終了時刻が入力されているか
   - 必要な理由欄に入力があるか

2. **患者情報が正しく設定されているか**
   - 生年月日が設定されているか（年齢計算が必要な加算の場合）
   - 死亡日・死亡場所が設定されているか（ターミナルケア加算の場合）
   - 建物情報が設定されているか（複数回訪問加算の場合）

3. **施設設定が有効になっているか**
   - 24時間対応体制加算や緊急時訪問看護加算は、施設設定で有効にする必要があります

4. **加算マスタが有効になっているか**
   - 加算マスタ管理画面で、該当する加算が有効になっているか確認してください
   - 訪問日が加算マスタの有効期間内か確認してください

### Q2. 点数が正しくないのはなぜですか？

**A.** 以下の点を確認してください:

1. **条件分岐タイプの加算の場合**
   - 訪問時間や訪問回数などの条件が正しく判定されているか確認してください
   - 訪問記録画面の「適用加算」セクションで、どの条件で加算が適用されたか確認できます

2. **加算マスタの点数設定**
   - 加算マスタ管理画面で、加算の点数が正しく設定されているか確認してください

### Q3. 同じ加算が複数回適用されることはありますか？

**A.** 基本的には、同じ訪問記録に同じ加算は1回のみ適用されます。ただし、以下の例外があります:

- **月次制限がある加算**: 同じ月内で複数の訪問記録に適用される場合があります（例: 緊急時訪問看護加算、専門管理加算）
- **条件が異なる加算**: 同じ加算でも、条件が異なれば別の加算として扱われる場合があります

### Q4. 加算を手動で追加・削除できますか？

**A.** 現在のバージョンでは、加算は自動計算のみです。手動での追加・削除はできません。

加算が適用されない場合は、上記のQ1を参考に、訪問記録や患者情報の設定を確認してください。

### Q5. 訪問記録を更新したら、加算が変わりました。なぜですか？

**A.** 訪問記録を更新すると、加算が自動的に再計算されます。以下の変更で加算が変わる可能性があります:

- 訪問日を変更した場合
- 訪問開始時刻・終了時刻を変更した場合
- チェックボックス（退院日当日の訪問、ターミナルケアなど）を変更した場合
- 理由欄（緊急訪問の理由、複数回訪問の理由など）を追加・削除した場合
