## レセプト一覧画面

レセプト一覧画面では、月次レセプトの一覧を確認し、レセプトの生成・確定・出力などの操作を行います。

### 画面の構成

レセプト一覧画面は、以下のセクションで構成されています:

| セクション | 説明 |
|-----------|------|
| ヘッダー | 画面タイトルと「レセプト生成」ボタン |
| 絞り込み | レセプトを検索・フィルタするための条件設定 |
| レセプト一覧 | レセプトの一覧表 |

### レセプトの生成

1. 画面右上の「レセプト生成」ボタンをクリックします
2. レセプト生成ダイアログが表示されます
3. 以下の項目を選択します:
   - **対象年**: レセプトを作成する年を選択します
   - **対象月**: レセプトを作成する月を選択します
   - **保険種別**: 「医療保険」または「介護保険」を選択します
4. 「生成」ボタンをクリックします
5. 対象年月の訪問記録から、自動的にレセプトが生成されます
6. レセプト生成時に、CSV出力の準備状況も自動的に確認・更新されます

**注意**: 同じ対象年月・保険種別のレセプトが既に存在する場合、重複して生成されることはありません。既存の未確定レセプトがある場合は、そのレセプトが更新されます。

#### レセプトが生成される条件

レセプトは、以下の**すべての条件**を満たす利用者についてのみ生成されます:

1. **訪問記録が存在すること**
   - 対象年月に、「完了」または「確認済み」の状態の訪問記録が1件以上存在していること
   - 「下書き」状態の訪問記録は対象外です

2. **保険証が登録されていること**
   - 利用者に、選択した保険種別（医療保険または介護保険）の保険証が登録されていること
   - 保険証の有効期間が対象年月と重複していること

3. **確定済みレセプトが存在しないこと**
   - 同じ対象年月・保険種別の確定済みレセプトが存在しないこと
   - 未確定のレセプトが既に存在する場合は、そのレセプトが更新されます

#### レセプトを確定・CSV出力するために必要な条件

レセプトを確定してCSV出力するためには、上記の生成条件に加えて、以下の条件も必要です:

1. **訪問看護指示書が登録されていること**
   - 利用者に、対象月の訪問記録の訪問日が有効期間内に含まれる訪問看護指示書が登録されていること
   - 訪問看護指示書には、以下の情報が入力されている必要があります:
     - 保険種別（医療保険または介護保険）
     - 指示区分（通常、退院時、退院後など）
     - ICD-10コード（病名コード、7桁以内の英数字）

2. **エラーがないこと**
   - レセプト詳細画面に表示されるエラーをすべて解消していること
   - エラーがある場合は、レセプトを確定できません

**補足**: レセプトは生成できても、訪問看護指示書が登録されていない場合は確定できません。CSV出力を行うためには、必ず訪問看護指示書を登録してください。

### 絞り込み機能

レセプト一覧を絞り込むには、以下の条件を設定します:

| 項目 | 説明 |
|------|------|
| 対象年 | 表示するレセプトの年を選択します（「すべて」を選択すると全期間が対象） |
| 対象月 | 表示するレセプトの月を選択します（「すべて」を選択すると全期間が対象） |
| 利用者 | 特定の利用者のレセプトのみを表示します（「すべて」を選択すると全利用者が対象） |
| 保険種別 | 「医療保険」または「介護保険」を選択します |
| ステータス | レセプトの状態で絞り込みます（以下のいずれかを選択） |

**ステータスの選択肢**:

| ステータス | 説明 |
|-----------|------|
| すべて | すべてのレセプトを表示 |
| エラーあり | エラーがあるレセプトのみ表示 |
| 警告あり | 警告があるレセプトのみ表示 |
| 未確定 | 未確定のレセプトのみ表示 |
| 確定済み | 確定済みのレセプトのみ表示 |
| 送信済み | 送信済みのレセプトのみ表示 |

### レセプト一覧表の項目

レセプト一覧表には、以下の項目が表示されます:

| 項目名 | 説明 |
|--------|------|
| 対象月 | レセプトの対象年月（例: 2025年11月） |
| 利用者 | レセプトの対象となる利用者の氏名 |
| 保険種別 | 「医療保険」または「介護保険」のバッジ |
| 訪問回数 | 対象月の訪問回数 |
| 合計点数 | レセプトの合計点数（医療保険は「点」、介護保険は「単位」） |
| 合計金額 | レセプトの合計金額（円） |
| ステータス | レセプトの状態を表すバッジ |
| CSV出力 | CSV出力の準備状況を表すバッジ |
| 操作 | レセプトに対する操作ボタン |

### ステータスの種類

レセプトのステータスは、以下の種類があります:

| ステータス | 説明 |
|-----------|------|
| 未確定 | レセプトが未確定の状態です。編集・確定が可能です |
| 警告あり | レセプトに警告がある状態です。確定は可能ですが、内容の確認を推奨します |
| エラーあり | レセプトにエラーがある状態です。エラーを修正するまで確定できません |
| 確定済み | レセプトが確定された状態です。編集はできませんが、再開して編集可能にできます |
| 送信済み | レセプトが送信された状態です。編集・再開はできません |

**補足**: レセプトを確定するには、エラーがないことが必要です。警告がある場合でも確定は可能ですが、内容を確認することをお勧めします。

### CSV出力の準備状況

CSV出力の準備状況は、以下のバッジで表示されます:

| バッジ | 説明 |
|--------|------|
| 準備完了 | CSV出力の準備が整っています |
| 不足あり | CSV出力に必要なデータが不足しています。レセプト詳細画面で確認してください |
| 未チェック | CSV出力の準備状況が未確認です |

**自動更新について**:

CSV出力の準備状況は、以下のタイミングで自動的に更新されます:

| タイミング | 説明 |
|-----------|------|
| レセプト生成時 | レセプトを生成した際に、自動的にCSV出力準備状況が確認・更新されます |
| レセプト確定時 | レセプトを確定した際に、自動的にCSV出力準備状況が確認・更新されます |
| レセプト再計算時 | レセプトを再計算した際に、自動的にCSV出力準備状況が確認・更新されます |
| 手動検証時 | レセプト詳細画面の「検証」ボタンをクリックした際に、CSV出力準備状況が確認・更新されます |

### 操作ボタン

レセプト一覧表の「操作」列には、レセプトの状態に応じて以下のボタンが表示されます:

#### 未確定のレセプトの場合

| ボタン | 説明 |
|--------|------|
| 再計算 | 訪問記録の加算を再計算して、レセプトの点数・金額を更新します。再計算時に、CSV出力の準備状況も自動的に確認・更新されます |
| 詳細 | レセプト詳細画面を開きます |
| 確定 | レセプトを確定します（エラーがある場合は確定できません）。確定時に、CSV出力の準備状況も自動的に確認・更新されます |

#### 確定済みのレセプトの場合

| ボタン | 説明 |
|--------|------|
| 詳細 | レセプト詳細画面を開きます |
| 再開 | レセプトを再開して編集可能にします（送信済みの場合は再開できません） |

### CSV出力（一括出力）

レセプト一覧画面の右上には、以下のボタンが表示されます:

| ボタン | 説明 |
|--------|------|
| 医療保険レセプトデータ出力 | 表示されている確定済みの医療保険レセプトを一括でCSV出力します |
| 介護保険レセプトデータ出力 | 表示されている確定済みの介護保険レセプトを一括でCSV出力します |

**注意**: 一括出力は、確定済みのレセプトのみが対象です。未確定のレセプトは出力されません。

## レセプト詳細画面

レセプト詳細画面では、個別のレセプトの詳細情報を確認し、編集・確定・出力などの操作を行います。

### 画面の構成

レセプト詳細画面は、以下のセクションで構成されています:

| セクション | 説明 |
|-----------|------|
| ヘッダー | レセプトの基本情報と操作ボタン |
| バリデーションエラー | エラーがある場合に表示されます |
| 確認推奨事項 | 警告がある場合に表示されます |
| 利用者情報 | レセプトの対象となる利用者の情報 |
| 保険証情報 | 利用者の保険証情報 |
| 訪問看護指示書 | 訪問看護指示書の情報 |
| 公費負担医療情報 | 公費負担医療の情報（該当する場合） |
| レセプト集計 | 訪問回数・点数・金額の集計情報 |
| 一部負担金額・減免情報 | 医療保険レセプトCSV出力用の情報（医療保険のみ） |
| 高額療養費適用状況 | 高額療養費の適用状況（医療保険のみ） |
| 加算情報 | 適用された加算とサービスコードの選択状況 |
| 訪問記録 | 対象月の訪問記録一覧 |
| 確定情報 | レセプトの確定日時・確定者（確定済みの場合） |

### ヘッダーの操作ボタン

レセプト詳細画面のヘッダーには、以下の操作ボタンが表示されます:

| ボタン | 説明 |
|--------|------|
| 一覧に戻る | レセプト一覧画面に戻ります |
| 医療保険レセプトCSV出力 | 医療保険レセプトのCSVファイルをダウンロードします（医療保険のみ） |
| 訪問看護療養費明細書Excel出力 | 訪問看護療養費明細書のExcelファイルをダウンロードします（医療保険のみ） |
| 検証 | レセプトの内容を検証して、エラーや警告を確認します。検証時に、CSV出力の準備状況も自動的に確認・更新されます |
| 確定 | レセプトを確定します（未確定の場合）。確定時に、CSV出力の準備状況も自動的に確認・更新されます |
| 再開 | レセプトを再開して編集可能にします（確定済みの場合） |

### バリデーションエラー

レセプトにエラーがある場合、画面の上部に「バリデーションエラー」セクションが表示されます。

エラーがある場合、レセプトを確定できません。エラーの内容を確認し、必要な情報を入力・修正してください。

### 確認推奨事項

レセプトに警告がある場合、画面の上部に「確認推奨事項」セクションが表示されます。

警告がある場合でもレセプトの確定は可能ですが、内容の確認を推奨します。

### 利用者情報

利用者情報セクションには、以下の情報が表示されます:

| 項目名 | 説明 |
|--------|------|
| 患者番号 | 利用者の番号 |
| 氏名 | 利用者の氏名 |
| 生年月日 | 利用者の生年月日 |
| 性別 | 利用者の性別 |
| 電話番号 | 利用者の電話番号 |
| 住所 | 利用者の住所 |

利用者情報は、「編集」ボタンをクリックして編集できます。

### 保険証情報

保険証情報セクションには、以下の情報が表示されます:

| 項目名 | 説明 |
|--------|------|
| 保険種別 | 「医療保険」または「介護保険」 |
| 保険者番号 | 保険者の番号 |
| 被保険者番号 | 被保険者の番号 |
| 自己負担割合 | 自己負担の割合（例: 3割） |
| 本人家族区分 | 本人・家族などの区分（該当する場合） |
| 年齢区分 | 未就学者・一般・高齢者などの区分（該当する場合） |
| 高齢受給者区分 | 高齢受給者の区分（該当する場合） |
| 有効期間 | 保険証の有効期間 |

保険証情報は、「編集」ボタンをクリックして編集できます。

### 訪問看護指示書

| 項目名 | 説明 |

| ICD-10コード | 病名のICD-10コード（該当する場合） |
| 点滴注射指示 | 点滴注射の指示の有無（該当する場合） |
| 床ずれ処置 | 床ずれ処置の有無（該当する場合） |

公費負担医療がある場合、公費負担医療情報セクションが表示されます。

各公費負担医療について、以下の情報が表示されます:

| 項目名 | 説明 |
|--------|------|
| 優先順位 | 公費負担医療の優先順位 |
| 法別番号 | 法別番号 |
| 負担者番号 | 負担者番号 |
| 受給者番号 | 受給者番号 |
| 有効期間 | 公費負担医療の有効期間 |
| 備考 | 備考（該当する場合） |

また、各公費負担医療について、以下の情報を入力できます（医療保険レセプトCSV出力用）:

| 項目名 | 説明 |
|--------|------|
| 一部負担金額（円） | 一部負担金額（8桁以内の数字、任意入力） |
| 公費給付対象一部負担金（円） | 公費給付対象一部負担金（6桁以内の数字、任意入力） |

**注意**: 
- 各公費カードごとに独立した入力フィールドが表示されます
- 入力フィールドからフォーカスが外れたときに自動保存されます
- 確定済みのレセプトは編集できません

### レセプト集計

レセプト集計セクションには、以下の情報が表示されます:

| 項目名 | 説明 |
|--------|------|
| 訪問回数 | 対象月の訪問回数 |
| 基本点数 | 基本療養費の点数合計 |
| 合計点数 | 基本点数と加算点数の合計 |
| 合計金額 | レセプトの合計金額（円） |

### 一部負担金額・減免情報（医療保険のみ）

医療保険レセプトの場合、一部負担金額・減免情報セクションが表示されます。

このセクションでは、医療保険レセプトCSV出力用の以下の情報を入力できます:

| 項目名 | 説明 |
|--------|------|
| 証明書番号 | 国民健康保険一部負担金減額・免除・徴収猶予証明書の証明書番号（国保の場合のみ表示、3桁以内の数字） |
| 一部負担金額（円） | 一部負担金額（8桁以内、任意入力） |
| 減免区分 | 減額・免除・支払猶予の区分（別表9: 1=減額、2=免除、3=支払猶予） |
| 減額割合（%） | 減額の場合の減額割合（減免区分が「減額」の場合のみ表示、0-100%） |
| 減額金額（円） | 減額の場合の減額金額（減免区分が「減額」の場合のみ表示、6桁以内） |

**注意**: 
- 証明書番号は国保の場合のみ表示されます
- 減額割合と減額金額は、減免区分で「減額」を選択した場合のみ表示されます
- 入力フィールドからフォーカスが外れたときに自動保存されます
- 選択項目（減免区分）は閉じたときに自動保存されます
- 確定済みのレセプトは編集できません

### 高額療養費適用状況（医療保険のみ）

医療保険レセプトの場合、高額療養費適用状況セクションが表示されます。

このセクションでは、以下の情報を選択できます:

| 項目名 | 選択肢 |
|--------|--------|
| 高額療養費適用状況 | 高額療養費該当なし<br>高額療養費現物給付あり（多数回該当を除く）<br>高額療養費現物給付あり（多数回該当） |

**注意**: 
- 医療保険レセプトのみ表示されます
- 選択項目は閉じたときに自動保存されます
- 確定済みのレセプトは編集できません
- CSV出力時に選択された値に応じてコードが出力されます（未選択の場合は「00」、多数回該当を除く場合は「01」、多数回該当の場合は「02」）

### 加算情報

加算情報セクションには、適用された加算の一覧が表示されます。

**注意**: 「下書き」状態の訪問記録の加算は表示されません。「完了」または「確認済み」の状態の訪問記録の加算のみが表示されます。

加算情報は、以下の2つのセクションに分かれています:

1. **サービスコード選択済みの加算**: サービスコードが選択されている加算
2. **サービスコード未選択の加算**: サービスコードが選択されていない加算

#### サービスコード選択済みの加算

サービスコードが選択されている加算には、以下の情報が表示されます:

| 項目名 | 説明 |
|--------|------|
| 訪問日 | 加算が適用された訪問記録の訪問日 |
| 訪問時間 | 訪問記録の訪問時間 |
| 加算名 | 加算の名称 |
| 点数 | 加算の点数（医療保険は「点」、介護保険は「単位」） |
| サービスコード | 選択されているサービスコード |

サービスコードを変更する場合は、ドロップダウンから別のサービスコードを選択します。

#### サービスコード未選択の加算

サービスコードが選択されていない加算には、以下の情報が表示されます:

| 項目名 | 説明 |
|--------|------|
| 訪問日 | 加算が適用された訪問記録の訪問日 |
| 訪問時間 | 訪問記録の訪問時間 |
| 加算名 | 加算の名称 |
| 点数 | 加算の点数（医療保険は「点」、介護保険は「単位」） |
| 加算コード | 加算のコード |

サービスコードを選択するには、ドロップダウンから適切なサービスコードを選択します。

**注意**: 確定済みのレセプトは編集できません。

### 訪問記録

訪問記録セクションには、対象月の訪問記録一覧が表示されます。

**注意**: 「下書き」状態の訪問記録は表示されません。「完了」または「確認済み」の状態の訪問記録のみが表示されます。

訪問記録一覧表には、以下の項目が表示されます:

| 項目名 | 説明 |
|--------|------|
| 訪問日 | 訪問記録の訪問日 |
| 訪問時間 | 訪問記録の訪問時間（開始時刻 - 終了時刻） |
| 担当看護師 | 訪問記録の担当看護師 |
| ステータス | 訪問記録の状態（完了・確認済み・下書きなど） |
| 適用加算 | 訪問記録に適用された加算の一覧 |

訪問記録の行をクリックすると、訪問記録詳細画面に遷移します。

### 確定情報

レセプトが確定済みの場合、確定情報セクションが表示されます。

確定情報セクションには、以下の情報が表示されます:

| 項目名 | 説明 |
|--------|------|
| 確定日時 | レセプトが確定された日時 |
| 確定者 | レセプトを確定したユーザーの氏名 |
| 送信日時 | レセプトが送信された日時（送信済みの場合） |

## よくある質問

### Q1. レセプトが生成されないのはなぜですか？

**A.** 以下の点を確認してください:

1. **訪問記録が存在するか**
   - 対象年月に、「完了」または「確認済み」の状態の訪問記録が1件以上存在するか確認してください
   - 「下書き」状態の訪問記録はレセプト生成の対象外です
   - 訪問記録が存在しない場合、レセプトは生成されません

2. **保険証が登録されているか**
   - 利用者に、選択した保険種別（医療保険または介護保険）の保険証が登録されているか確認してください
   - 保険証の有効期間が対象年月と重複しているか確認してください
   - 保険証が登録されていない場合、または有効期間が対象年月と重複していない場合、レセプトは生成されません

3. **確定済みレセプトが存在しないか**
   - 同じ対象年月・保険種別の確定済みレセプトが既に存在する場合、レセプトは生成されません
   - 未確定のレセプトが存在する場合は、そのレセプトが更新されます

### Q2. レセプトを確定できないのはなぜですか？

**A.** 以下の点を確認してください:

1. **エラーがないか**
   - レセプト詳細画面の「バリデーションエラー」セクションを確認してください
   - エラーがある場合、エラーの内容を確認し、必要な情報を入力・修正してください
   - エラーがある場合は、レセプトを確定できません

2. **訪問看護指示書が登録されているか**
   - 利用者に、対象月の訪問記録の訪問日が有効期間内に含まれる訪問看護指示書が登録されているか確認してください
   - 訪問看護指示書が登録されていない場合、または訪問日が指示書の有効期間外の場合、エラーが表示されます

3. **訪問看護指示書の必須項目が入力されているか**
   - 訪問看護指示書に、以下の情報が入力されているか確認してください:
     - 保険種別（医療保険または介護保険）
     - 指示区分（通常、退院時、退院後など）
     - ICD-10コード（病名コード、7桁以内の英数字）
   - これらの情報が未入力の場合、エラーまたは警告が表示されます

### Q3. CSV出力ができないのはなぜですか？

**A.** 以下の点を確認してください:

1. **レセプトが確定済みか**
   - CSV出力は、確定済みのレセプトのみが対象です
   - 未確定のレセプトは、先に確定してください

2. **CSV出力の準備状況**
   - レセプト一覧画面の「CSV出力」列を確認してください
   - 「不足あり」と表示されている場合、レセプト詳細画面で不足しているデータを確認してください
   - CSV出力準備状況は、レセプト生成・確定・再計算・検証時に自動的に更新されます
   - 「未チェック」と表示されている場合は、レセプト詳細画面の「検証」ボタンをクリックして確認してください

3. **訪問看護指示書が登録されているか**
   - 利用者に、対象月の訪問記録の訪問日が有効期間内に含まれる訪問看護指示書が登録されているか確認してください
   - 訪問看護指示書に、保険種別、指示区分、ICD-10コードなどの必須情報が入力されているか確認してください

4. **医療保険レセプトの場合**
   - 一部負担金額・減免情報、高額療養費適用状況など、必要な情報が入力されているか確認してください
   - 加算のサービスコードが選択されているか確認してください

### Q4. 加算のサービスコードを選択する必要がありますか？

**A.** 医療保険レセプトのCSV出力を行う場合、加算のサービスコードを選択する必要があります。

サービスコードが選択されていない加算がある場合、レセプト詳細画面の「サービスコード未選択の加算」セクションに表示されます。

各加算について、適切なサービスコードを選択してください。

### Q5. レセプトを再開できますか？

**A.** 確定済みのレセプトは、再開して編集可能にできます。

ただし、以下の場合は再開できません:

- レセプトが送信済みの場合
- レセプトが確定されていない場合（未確定のレセプトは再開の必要がありません）

再開するには、レセプト詳細画面の「再開」ボタンをクリックします。

### Q6. レセプトの点数が正しくない場合はどうすればよいですか？

**A.** 以下の手順で確認してください:

1. **訪問記録の加算を再計算**
   - レセプト一覧画面で「再計算」ボタンをクリックします
   - 訪問記録の加算が再計算され、レセプトの点数・金額が更新されます

2. **訪問記録を確認**
   - レセプト詳細画面の「訪問記録」セクションで、訪問記録の内容を確認してください
   - 訪問記録に適用された加算が正しいか確認してください

3. **加算のサービスコードを確認**
   - レセプト詳細画面の「加算情報」セクションで、加算のサービスコードが正しく選択されているか確認してください
